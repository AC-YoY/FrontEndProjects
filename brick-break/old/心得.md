##### Q.左右移动画面抖动

```
原因：
浏览器默认的keydown触发数量不够，fps不够，导致画面抖动
解决：
原方法设置标记，使用 setInterval 触发实际事件
var flag
setInterval( function () {
    if ( flag ) {
        doSomthing()
    }
}, 1000/30 )
```

##### 注册事件的写法
```
// 原本 keydown keyup 事件里面判断每一个按键 具体的操作
windown.addEventListener( 'keydown', function( e ) {
    if ( e.key === 'a' ) {
        executeAMethod()
    }
} )
// 逻辑写成注册事件
//  1. 保存每个按键的状态
obj = {
    keydowns: [],
    actions: []
}
windown.addEventListener( 'keydown', function( e ) {
    obj.keydowns[ e.key ] = true
} )
//  2. 注册事件
obj.registerAction = function( key, callback ) {
    if ( obj.keydowns[ key ] ) {
        callback()
    }
}
//  3. 使用
obj.register( 'a', function() {
    // 执行你要的事件
} )
```
